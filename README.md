# AI Learning Buddy

## Description
AI Learning Buddy is a Streamlit-based web application developed as part of the Infosys EmpowerHer assignment. The application helps learners understand concepts through AI-generated explanations, real-life examples, quizzes, and feedback.

## Features
- Explain topics in simple language
- Generate real-life examples
- Create multiple-choice quizzes
- Evaluate learner answers with AI-generated feedback
- Simple and user-friendly interface

## Technologies Used
- Python
- Streamlit
- Google Gemini AI

## Project Topic
Binary Search

## How to Run the Application

1. Install the required dependencies:
```bash
pip install -r requirements.txt
```

2. Run the Streamlit application:
```bash
streamlit run app.py
```
